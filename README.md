# 💳 Pasarela de Pagos Flutter

Aplicación desarrollada en **Flutter** que simula el proceso de compra y pago de productos mediante tarjeta bancaria. Permite seleccionar productos, ingresar datos de pago, validar la información ingresada y visualizar el resultado de la transacción de forma clara e intuitiva.

---

## 👩‍💻 Autor

**Nayely Ayol**
**Alessia Pérez**

---

## 🎯 Objetivo

* Desarrollar una aplicación móvil utilizando Flutter.
* Simular el flujo de una pasarela de pagos.
* Validar la información ingresada por el usuario.
* Gestionar pagos de forma visual e interactiva.
* Mostrar el historial de transacciones realizadas.
* Aplicar diseño responsive para diferentes dispositivos.

---

## ✨ Funcionalidades

✅ Visualización de productos

✅ Selección de productos para compra

✅ Ingreso de credenciales y datos de pago

✅ Validación de campos obligatorios

✅ Resumen de compra

✅ Historial de pagos

✅ Confirmación de pago aceptado

✅ Confirmación de pago rechazado

✅ Navegación entre pantallas

✅ Diseño responsive

✅ Compatible con Android, Web y Windows
